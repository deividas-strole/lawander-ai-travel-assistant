# AI Travel Planner by Deividas Strole

**AI Travel Planner** is a smart web application built with **Spring Boot** and **React** that helps travelers plan their trips efficiently. Developed by **Deividas Strole**, this app generates personalized travel itineraries and displays points of interest on an interactive map using AI-powered suggestions.

---

## 🚀 About the Project

This AI Travel Planner combines modern **frontend** and **backend** technologies to provide a seamless travel planning experience. Users can input a destination and number of days, and the app generates a day-by-day itinerary highlighting top attractions, restaurants, hotels, parks, and cultural sites.

The app is designed and developed by **Deividas Strole** to showcase expertise in **full-stack development**, AI integration, and interactive map visualizations.

---

## 💡 Features

- AI-generated, **personalized travel itineraries** for any destination.
- Interactive **map with custom markers** for destinations, attractions, hotels, restaurants, and parks.
- Highlights **top tourist spots, cultural sites, and dining options**.
- Supports clickable markers with popups containing detailed information.
- Easy-to-use chat interface for asking destination-specific questions.
- Automatic geocoding and filtering of attractions within 50 km radius.

---

## 🛠️ Technologies Used

- **Frontend:** React, React-Leaflet, Leaflet, HTML, CSS  
- **Backend:** Spring Boot, Java  
- **APIs:** OpenStreetMap Nominatim for geocoding  
- **AI Integration:** Generates dynamic itineraries and recommendations  
- **Other:** JavaScript ES6+, responsive UI design

---

🎯 Usage

1. Enter your destination and number of days.
2. AI generates a detailed itinerary with top attractions.
3. Explore locations on the interactive map.
4. Click markers for detailed popups about each place.
5. Ask questions in the chat to get custom recommendations.

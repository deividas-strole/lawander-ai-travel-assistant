ğŸ“¦ Installation & Setup

**Prerequisites**

Make sure you have the following installed:

- Java JDK 17+
- Node.js 16+ and npm
- PostgreSQL 13+
- Maven (or use the provided wrapper)

**Steps**

1. Clone the repository
2. Set up PostgreSQL database and update `application.properties` with your credentials
3. Start the backend: `cd backend && ./mvnw spring-boot:run`
4. Start the frontend: `cd frontend && npm install && npm start`
